# Roadmap

## Phase 1

- Google-only auth via Supabase; admin bootstrap; IsAllowed gating.
- CRUD servers with Docker containers; Paper/Fabric/Spigot/Bukkit support.
- Dynamic version resolution (latest or specific).
- Basic container metrics and log tailing.
- Read-only file browser scoped to server root.

## Phase 2

- Optional in-server plugin/mod for rich metrics.
- Backups and restores; scheduled restarts.
- Alerting basics (crash/high memory).

## Phase 3

- Multi-tenant hardening; granular roles/permissions.
- Advanced observability and integrations.
- Cluster/remote host support.
